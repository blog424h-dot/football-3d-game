# سياسة الخصوصية للعبة كرة القدم 3D

**1. جمع المعلومات:**  
هذه اللعبة لا تجمع أي بيانات شخصية من المستخدمين. كل البيانات المتعلقة باللعبة محلية فقط على جهاز المستخدم (مثل التشكيلة والنتيجة).

**2. الاستخدام:**  
اللعبة مخصصة للترفيه فقط. أي بيانات مؤقتة لا تُخزن على أي خادم.

**3. ملفات الطرف الثالث:**  
اللعبة تستخدم مكتبات مفتوحة المصدر مثل Three.js، ولا تجمع هذه المكتبات أي معلومات عن المستخدمين.

**4. التواصل:**  
إذا كانت لديك أسئلة حول الخصوصية أو استخدام اللعبة، يمكنك التواصل عبر صفحة GitHub الخاصة بالمستودع.

**5. الأطفال:**  
اللعبة آمنة للأطفال ولا تتطلب أي تسجيل أو بيانات شخصية.

**6. التغييرات:**  
قد يتم تحديث هذه السياسة عند إضافة ميزات جديدة، وسيتم توضيح التغييرات في صفحة المستودع.
